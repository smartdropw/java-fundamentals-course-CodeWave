# Curso de Fundamentos de Java para Escolares

Este repositorio contiene el código fuente desarrollado para el curso introductorio de programación en Java, dirigido a estudiantes de secundaria (12 a 17 años).

## Estructura del Repositorio

* `/starter-files`: Contiene las plantillas de código iniciales para que los estudiantes practiquen.
* `/completed-examples`: Contiene las soluciones completas de los códigos explicados en cada video.
* `/scripts`: Archivos complementarios del proyecto.

## Instrucciones de Configuración y Uso

Para ejecutar y probar los ejemplos de este curso, no es necesario descargar ni instalar ningún software (como un IDE o el JDK) en tu computadora local. Todo el contenido está diseñado para ser renderizado y ejecutado directamente en la web.

1. Dirígete a la carpeta `/completed-examples` de este repositorio.
2. Selecciona el archivo de la lección que deseas revisar y copia su contenido.
3. Abre el editor en línea [OnlineGDB](https://www.onlinegdb.com/).
4. En la esquina superior derecha del editor, asegúrate de seleccionar el lenguaje **Java**.
5. Pega el código en la sección central y haz clic en el botón verde **Run** para ejecutar la aplicación.
